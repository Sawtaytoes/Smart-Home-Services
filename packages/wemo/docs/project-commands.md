# Project Commands

Default runner (same as yarn api:watch)
```shell
yarn api
```

Run the project
```shell
yarn api
```

Run with Node.js debugging
```shell
yarn api:debug
```

Restart when files change
```shell
yarn api:watch
```

Run with Node.js debugging and watch for changes
```shell
yarn api:watch:debug
```

Run unit tests
```shell
yarn test
```

Run unit tests and watch for changes
```shell
yarn test:watch
```
