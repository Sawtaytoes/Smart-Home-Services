# Running Flic Daemon

## Development
```shell
sudo ~/projects/fliclib-linux-hci/bin/armv6l/flicd -f flic-db.sqlite3 -w -s 0.0.0.0
```

## Production
```shell
sudo ~/projects/fliclib-linux-hci/bin/armv6l/flicd -f flic-db.sqlite3 -w -d
```
